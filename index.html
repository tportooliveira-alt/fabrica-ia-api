Bem-vindo ao IdeaOrganizer (Fábrica Autônoma)

Esta não é apenas uma ferramenta de anotações. É um **Orquestrador de Agentes IA** projetado para refinar suas ideias e gerar pacotes de código prontos para o Antigravity e MCPs (Claude Code / Gemini Code).

1️⃣

Capture a Ideia

Escreva ou grave um áudio com um simples prompt. O sistema entende o contexto e cria um projeto base.

2️⃣

Orquestração Dinâmica

O Arquiteto Mestre no backend (node) decide quais especialistas (Design, Backend, BD) precisam debater sobre a sua ideia usando os Skills locais.

3️⃣

Tradutor Antigravity

Ao final da fábrica, você recebe um Blueprint Master e comandos prontos para plugar direto no seu CLI do Antigravity para iniciar a codificação autônoma.

Use o campo abaixo para capturar sua primeira ideia e iniciar o processo.
